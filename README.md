## 作品演示视频

### 文化遗迹数字博物馆（Cultural Relics Museum）

基于 Unity3D 引擎独立开发的沉浸式 3D 虚拟博物馆，包含第三人称角色探索、文物展览、拼图解谜系统、类原神传送系统等完整模块。22+ 件中国文物独立 3D 展示，支持 4K 分辨率。

### 教室VR交互系统

基于 Unity URP + XR Interaction Toolkit 开发的 VR 教室交互系统。阶段性顺序放置引导（SequenceSpawner + SocketMatcher）、纸团物理抓取投掷、URP 管线实时光照兼容调试。

### 模块化塔防（Modular Tower Defense）⚠️ 开发中

> **注意：此项目为半成品，核心玩法仍在开发中。** 当前展示为阶段性成果，不代表最终完成度。

基于 Unity 引擎的 3D 模块化塔防游戏，核心创新为「乐高式炮塔组装」——底座、武器、增效器三层自由搭配。包含工坊设计、蓝图管理、科技树、波次战斗等系统。架构上采用 6 层程序集隔离 + 事件总线解耦 + ScriptableObject 数据驱动。
